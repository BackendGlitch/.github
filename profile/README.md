Backend Glitch builds backend infrastructure, AI systems, and blockchain products designed for the Tunisian and North African market. We do not wrap foreign APIs. We write the underlying software.

### Products

| Product | What it is |
|---------|------------|
| **Wasla** | AI-powered louage station management. Computer vision for vehicle detection and license plate recognition. Live queue management. Deployed in production. |
| **NexaPay** | Blockchain e-wallet on a custom L1 chain. KYC verification pipeline with bilingual CIN support. On-chain identity attestation. |
| **Fabrix** | Distributed 3D printing marketplace. Central orchestration server matching print jobs to makers across Tunisia. |
| **Tanit** | Identity verification infrastructure. Live KYC portal with real-time document detection, liveness checks, and trust scoring. |

### Stack

Rust, Go, Python, TypeScript. PostgreSQL, Redis, Docker. PyTorch, ONNX, OpenCV, PaddleOCR. Solana, EVM, custom L1. We pick the right tool for the problem and we do not chase trends.

### Open Source

Most of our work is public. Every product has a reference implementation. Every line of code is written for someone who was not in the room.
